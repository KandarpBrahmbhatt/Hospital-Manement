# Hospital Management System

A modern and scalable Hospital Management System built with Node.js, Express.js, MongoDB, and TypeScript. The system provides comprehensive healthcare management solutions for handling patients, doctors, appointments, billing, insurance, emergency cases, and hospital analytics.

## Features

* Patient Management
* Doctor Management
* Department Management
* Appointment Scheduling
* Billing Management
* Insurance Management
* Emergency Case Handling
* Patient Activity Tracking
* MongoDB Aggregation Analytics
* Revenue Reports
* Doctor Performance Reports
* Patient Statistics
* RESTful APIs
* Database Indexing for Performance
* Bulk Data Seeder for Large Datasets
